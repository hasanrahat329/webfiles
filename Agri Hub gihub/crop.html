Crop Details

Essential information about common crops and their production time

Rice (ধান)

Scientific Name: Oryza sativa
Crop Type: Food Crop
Season: Kharif
Variety: BRRI Dhan-28, BRRI Dhan-29
Production Time: 120–150 days

Wheat (গম)

Scientific Name: Triticum aestivum
Crop Type: Food Crop
Season: Rabi
Variety: BARI Gom-30
Production Time: 110–130 days

Corn (ভুট্টা)

Scientific Name: Zea mays
Crop Type: Food & Cash Crop
Season: Kharif / Rabi
Variety: Hybrid Maize
Production Time: 90–110 days

Potato (আলু)

Scientific Name: Solanum tuberosum
Crop Type: Vegetable
Season: Rabi
Variety: Diamant, Cardinal
Production Time: 90–120 days

Tomato (টমেটো)

Scientific Name: Solanum lycopersicum
Crop Type: Vegetable
Season: Rabi / Summer
Variety: Roma, Hybrid Tomato
Production Time: 70–90 days

Onion (পেঁয়াজ)

Scientific Name: Allium cepa
Crop Type: Vegetable
Season: Rabi
Variety: Taherpuri, Hybrid Onion
Production Time: 100–120 days

Cabbage (বাঁধাকপি)

Scientific Name: Brassica oleracea
Crop Type: Vegetable
Season: Rabi
Variety: Golden Acre
Production Time: 80–100 days

Cauliflower (ফুলকপি)

Scientific Name: Brassica oleracea var. botrytis
Crop Type: Vegetable
Season: Rabi
Variety: Snowball, Hybrid
Production Time: 90–110 days

Chili (মরিচ)

Scientific Name: Capsicum annuum
Crop Type: Vegetable
Season: Kharif
Variety: Hybrid Chili
Production Time: 90–120 days

Mustard (সরিষা)

Scientific Name: Brassica juncea
Crop Type: Oilseed Crop
Season: Rabi
Variety: BARI Sarisha-6
Production Time: 85–95 days

Sugarcane (আখ)

Scientific Name: Saccharum officinarum
Crop Type: Cash Crop
Season: Kharif
Variety: Co 0238, BO 91
Production Time: 10–14 months

Jute (পাট)

Scientific Name: Corchorus olitorius
Crop Type: Fiber Crop
Season: Kharif
Variety: Tossa, White Jute
Production Time: 120–150 days
